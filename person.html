Person Core Vocabulary Holding Page

This is a holding page for the Person Core Vocabulary namespace document.

The Person Core Vocabulary is due to become a work item for the Government Linked Data WG in the very near future at which point we expect to publish the namspace document in various formats at this URI (http://www.w3.org/ns/person#). NB The current draft of the GLD's Terms for describing people for which the ISA work is an input uses the slightly different namespace of .../person. Only one (person or people) is expected to become the stable namespace (with a permanent redirect if necessary).

The Person Core Vocabulary has been developed under the ISA Programme of the European Union and all documentation, including the current version of the namespace document, is available on the European Commission's Joinup Platform.

For background, see this blog post: Interoperable Governments?.
