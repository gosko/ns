The Shape Constraint Langauge (SHACL) Vocabulary

SHACL Ontology

This document will be an HTML representation of an OWL ontology describing all vocabulary URIs defined in the Shape Constraint Langauge (SHACL) namespace. This ontology provides an informal representation of the concepts and terms as defined in the SHACL specification. Consult the SHACL specification for normative reference. See also http://www.w3.org/2012/shacl, http://www.w3.org/TR/shacl-ucr/, http://www.w3.org/TR/shacl/, http://www.w3.org/2011/09/LinkedData/

Classes are marked with the CSS class class, properties are marked with the CSS class prop, individuals are marked with the CSS class individual, and deprecated terms are marked with the CSS class deprecated.
