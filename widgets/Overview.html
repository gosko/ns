Widgets namespace

The namespace name http://www.w3.org/ns/widgets is intended for use with the Widgets 1.0 Specification.

This page is maintained by the Web Application Formats Working Group (charter), part of the Rich Web Clients Activity in the W3C Interaction Domain. Please send comments to public-appformats@w3.org, the W3C's public email list for issues related to Web Application Formats. Archives of the mailing list are available.

For more information about XML namespaces, please refer to Namespaces in XML.
