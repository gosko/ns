Web Services Description Language (WSDL) Version 2.0 Operation Safety

The Web Services Description Language (WSDL) Version 2.0 Operation safety extension is intended for use with the Web Services Description Language Version 2.0 (WSDL 2.0).

For a complete definition of the Operation safety, refer to Web Services Description Language (WSDL) Version 2.0 Part 2: Adjuncts specification.
