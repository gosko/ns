Linked Data Platform Namespace

This document describes the terms used in the Linked Data Platform. The URI of the vocabulary is http://www.w3.org/ns/ldp# (abbreviated by ldp: in this document). Turtle and an RDF/XML versions of the vocabulary are also available. The vocabulary is published by W3C

For a precise definition of these vocabulary elements, see the Linked Data Platform specification.

The vocabulary defines three classes :

  • ldp:Container: a @@ (see LDP §5. Linked Data Platform Container).
  • ldp:Page: a @@ (see LDP §5.1.3 Paging).

Properties

  • ldp:membershipSubject: indicates the subject of the membership triples when container subject is not the LDPC itself (see §5.2.5); it is used on ldp:Container.
  • ldp:membershipPredicate: indicates the predicate of the membership triple when the container predicate is not rdfs:member (see §5.2.5); it is used on ldp:Container.
  • ldp:pageOf: indicates the container in which the subject is a page (see §5.3.6.1); it is used on ldp:Page.
  • ldp:nextPage: the URL for the subsequent page (see §5.3.6.2); it is used on ldp:Page.
  • ldp:containerSortPredicates: a list of LDPC ordinal predicates (see LDP §5.1.4 Ordering); it is used on ldp:Page.

Warning: Until the specification reaches W3C Candidate Recommendation (CR) status, this namespace name may be reused by any update in such a way as to cause documents written using the namespace to become invalid or to change in meaning.

This document is intended to comply with W3C policy for XML namespace names.
