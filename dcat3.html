Status of this document

This vocabulary was published by the Dataset Exchange Working Group. See the full Data Catalog Vocabulary Version 3 specification for a detailed, human readable version of the vocabulary. Any comments or errata for this page or associated resources should use the process defined in that document.

Namespace URI

The namespace URI for DCAT is http://www.w3.org/ns/dcat#.

Alternative serialisations

The vocabulary is published both in human- and machine-readable forms. A GET request made to the canonical namespace URI will cause HTTP Content Negotiation to occur, which means that the most appropriate variant will be served according to your request. However, if you explicitly wish to retrieve a particular serialisation, you can also do so using the following URLs:

dcat3.html
HTML (this document)
dcat3.ttl
RDF (Turtle)
dcat3.rdf
RDF/XML
dcat3.jsonld
RDF (JSON-LD)
In addition, the RDF for externally defined terms where DCAT has provided additional documentation or usage notes can be found at dcat-external.ttl and alternatively dcat-external.rdf (XML/RDF) or dcat-external.jsonld (JSON-LD).

Older Versions of Ontology files

Version 3 aims to be backwardly compatible with this earlier versions.

  • The ontology for DCAT 2 can be seen in Turtle or RDF/XML or JSON-LD. The RDF for externally defined terms where DCAT has provided additional documentation or usage notes can be found at dcat2-external.ttl and alternatively dcat2-external.rdf (XML/RDF) or dcat2-external.jsonld (JSON-LD).
  • The ontology for the 2014 version of DCAT can be seen in Turtle or RDF/XML.
