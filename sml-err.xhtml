Service Modeling Language, Version 1.1 Namespaces

The namespace identified by the URI http://www.w3.org/ns/sml-err is not in use. A corresponding namespace was defined in an early draft (August 2007) of the Service Modeling Language, Version 1.1, but was dropped from later versions of the specification.

Any updates or changes in the use or non-use of this namespace will be documented here.

For information on the Service Modeling Language, Version 1.1, and its namespaces, see

  • Service Modeling Language, Version 1.1 (Candidate Recommendation draft of 25 November 2008).
  • Service Modeling Language Interchange Format, Version 1.1 (Candidate Recommendation draft of 25 November 2008).
  • the namespace document for the namespace http://www.w3.org/ns/sml
  • the namespace document for the namespace http://www.w3.org/ns/sml-function
  • the namespace document for the namespace http://www.w3.org/ns/sml-if
