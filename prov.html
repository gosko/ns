PROV Namespace

The namespace name http://www.w3.org/ns/prov# is intended for use with the set of PROV specifications. The terms defined within this namespace are identified in these specifications.

  • The PROV Data Model
  • The PROV Ontology
  • PROV-AQ: Provenance Access and Query

Warning: Until the specification reaches W3C Candidate Recommendation (CR) status, this namespace name may be reused by any update in such a way as to cause documents written using the namespace to become invalid or to change in meaning.

This document is intended to comply with W3C policy for XML namespace names.
