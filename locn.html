Location Core Vocabulary Holding Page

This is a holding page for the Location Core Vocabulary namespace document.

The Location Core Vocabulary is currently under the control of the Locations and Addresses Community Group. Should that group, or another group, be willing and able to develop and maintain it, then the schema can be published here.

The general forum for discussion of vocabularies at W3C is the Web Schemas Task Force, part of the Semantic Web Interest Group.

The Location Core Vocabulary was developed under the ISA Programme of the European Union and all documentation, including the current version of the namespace document, is available on the European Commission's Joinup Platform.

For background, see this blog post: Interoperable Governments?.
