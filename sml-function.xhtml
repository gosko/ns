Service Modeling Language Version 1.1 Function Namespace

The namespace identified by the URI http://www.w3.org/ns/sml-function is defined in the Service Modeling Language Version 1.1. This document contains a directory of links to resources related to that namespace, using Resource Directory Description Language.

Normative specification

This namespace is normatively defined by the current version of the specification Service Modeling Language Version 1.1.

At the time this namespace document was prepared, this namespace contained just the one name deref, which is the name of a function defined by the SML specification for use in XPath expressions.

Schema

There is no schema for this namespace.

SML and SML-IF Namespace Policy

The SML and SML-IF specifications define the following namespaces.

  • http://www.w3.org/ns/sml
  • http://www.w3.org/ns/sml-function
  • http://www.w3.org/ns/sml-if

Names in the namespaces listed above are defined in version 1.1 of the SML and SML-IF specifications, and may also be defined in future versions of the two specifications. The specifications define language extension mechanisms and how to handle changes such as the addition of new terms to the language. W3C reserves the right to determine which changes (backward compatible or not) are in the interest of the community at large.
