

See http://www.iana.org/assignments/media-types/media-types.xhtml
and http://www.iana.org/assignments/media-types/media-types.xml

Any Media type in those registries is deemed to correspond to a RDF Class
whose URI is that of this directory concatenated with that media type.
For example, a fixed content-type resource of MIME type 'image/png' is
deemed to be a member of the class

https://www.w3.org/ns/iana/media-types/image/png

This web space does not currently contain a copy of the IANA data or a mirror
of it, or a proxy of it but may do at a later date.
