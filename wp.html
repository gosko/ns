Abstract

Vocabulary document, specified by the W3C Publishing Working Group, defining the classes, properties, and individuals to be used by the Web Publication Manifest.

Status of This Document

This document is merely a W3C-internal document. It has no official standing of any kind and does not represent consensus of the W3C Membership.

This is a vocabulary specification for the Web Publication Manifest, defined by the W3C Publishing Working Group. The formal specifications of the terms, their usage, etc., are normatively defined by the specification for Web Publications.
