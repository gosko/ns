Unique identifier for the N3 file format.

'http://www.w3.org/ns/formats/N3' is the URI. N3 is defined by the “Notation3 (N3): A readable RDF syntax” W3C Team Submission. It has a media type: ‘text/rdf+n3’. (See the for further details on the Media Registration.)

Ivan Herman, ivan@w3.org, W3C, Semantic Web Activity Lead, 2010-05-03
