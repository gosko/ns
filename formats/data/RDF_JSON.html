Unique identifier for the RDF 1.1 JSON Alternate Serialization (RDF/JSON).

'http://www.w3.org/ns/formats/RDF_JSON' is the URI. RDF/JSON is defined by the “RDF 1.1 JSON Alternate Serialization (RDF/JSON)” W3C Working Group Note. It has a media type: ‘application/rdf+json’ and a preferred suffix: ‘.rj’. (See further details on the Media Registration.)
