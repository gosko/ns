Unique identifier for the RIF XML file format.

'http://www.w3.org/ns/formats/RIF_XML' is the URI. The RIF XML syntax is defined by the “RIF Core Dialect” Proposed W3C Recommendation. It has a media type: ‘application/rif+xml’ and a preferred suffix: ‘.rif’. (See further details on the Media Registration.)

Ivan Herman, ivan@w3.org, W3C, Semantic Web Activity Lead, 2010-05-03
