Unique identifier for the RDF/XML file format.

'http://www.w3.org/ns/formats/RDF_XML' is the URI. RDF/XML is defined by the “RDF/XML Syntax Specification” W3C Recommendation. It has a media type: ‘application/rdf+xml’ and a preferred suffix: ‘.rdf’. (See further details on the Media Registration.)

Ivan Herman, ivan@w3.org, W3C, Semantic Web Activity Lead, 2010-05-03
